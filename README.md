Nebula
======

3D Game Engine in C++

## Components

[core](http://github.com/chuck1/Nebula-Core)

## Dependencies

[glfw](http://github.com/glfw/glfw)
[glm](http://github.com/g-truc/glm)

## Installation

    git clone github.com/chuck1/Nebula.git
    cd Nebula
    git submodule init
    git submodule update
    mkdir Debug
    cd Debug
    cmake ..
    make install
