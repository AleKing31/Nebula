nebula
======

3d game engine in c++

[online documentation](http://chuck1.github.io/nebula/html/index.html)

## installation

requires [GRU](http://github.com/chuck1/GRU)

    git clone github.com/chuck1/nebula.git
    cd nebula
    mkdir build
    cd build
    cmake ..
    make install
    make doc
