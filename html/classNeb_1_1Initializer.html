Detailed Description

template<class T>
class Neb::Initializer< T >

Initializer. Store and use initializer functions. Allocate an object with arbitrary alloc function stored in the FuncMap and call the object's init method with arbitrary arguments.

Motivation:
create an object guaranteeing virtual initialization
Warning
T must have an init member with arguments passed to alloc which must be accessible from Initializer
